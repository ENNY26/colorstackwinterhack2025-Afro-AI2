# Afro AI 🌍

A voice-based language learning app for African languages. Practice real conversations with an AI tutor that speaks back to you in 8+ African languages.

## Features

- **Voice Conversations**: Speak and listen - have natural conversations in Yoruba, Swahili, Hausa, Zulu, and more
- **Cultural Context**: Real-time tips on proper phrases and cultural nuances during conversations
- **Smart Assistance**: Stuck? Get AI-generated response suggestions to keep the conversation flowing
- **Vocabulary Tracking**: Automatically logs key words and phrases from every conversation
- **Personality Selection**: Choose your AI tutor's tone (quirky, serious, friendly, etc.)

## Tech Stack

- **Frontend**: React Native / Expo
- **Speech-to-Text**: OpenAI Whisper API
- **Text-to-Speech**: ElevenLabs / Google Cloud TTS
- **AI Agent**: Claude API / GPT-4
- **Backend**: Firebase / Node.js

## Supported Languages

Yoruba • Swahili • Hausa • Zulu • Twi • Igbo • Xhosa
